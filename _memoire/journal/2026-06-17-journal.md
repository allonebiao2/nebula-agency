## Session 2 (soir) — Notifications stylées + outils de lancement partenaires

### A. Notifications « quand ça bouge » (demande Mongazi)
L'ossature existait (cloche + badge + polling `/api/signals` 12 s) mais **un seul son générique** (`ding`) pour tout. Refonte :
- **Notifs typées côté serveur** : colonne `notifs.kind` (migration ALTER TABLE) + param `kind` sur `notify()`. Chaque événement taggé : `client` (nouveau lead), `vente` (commissions générées), `recrue` (recrue/candidature/validation), `commission` (à réclamer / réclamation), `paiement` (versé), `statut` (changement de statut).
- **`/api/signals` renvoie `notif_top`** (id + kind + text du dernier non-lu) → le front choisit le son.
- **Tonalités Web Audio distinctes** (`static/app.js`, dispatcher `sound.notif(kind)`) : `client` = arpège cristallin montant · `vente` = triade lumineuse · `recrue` = « bienvenue » chaud · `commission/paiement` = cash · `statut` = blip · `message` = pop doux · fallback `ding`.
- **Fenêtre de notif enrichie** (admin + partenaire) : en-tête « X non lues · Y au total », **pastille de couleur par type** (`.ndot`), **badge qui pulse** (`badgePop`/`badgePulse`), **cloche qui sonne** (`.bell-ring` + `ringBell()`).

### B. Éléments manquants pour lancer & partager (2e demande)
Analyse → 2 vrais trous opérationnels comblés :
- **Renvoyer / réinitialiser le PIN** : le PIN est haché → irrécupérable dès la modale fermée. Nouveau endpoint `POST /api/admin/affiliates/{id}/reset-pin` (régénère un PIN, notifie le partenaire) + bouton **« Renvoyer accès »** par partenaire dans `/admin`. *Testé : ancien PIN invalidé, nouveau OK.*
- **Kit de bienvenue 1-tap** : helper `welcomeMessage()` + `renderAccessKit()` → carte d'accès commune (création, validation candidature, renvoi d'accès) montrant **espace `/partenaire` + Code + PIN + lien à partager `/p/CODE`**, avec **bouton « Envoyer sur WhatsApp »** (wa.me message pré-rempli, **sans destinataire forcé** — numérotation Bénin ambiguë, règle « ne jamais toucher un lien WhatsApp sans validation ») + « Copier le kit ».
