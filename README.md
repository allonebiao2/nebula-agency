# Nebula Agency

Agence web personnelle — vitrines modernes pour artisans, créateurs et petites marques.

## Structure

- **`00-nebula-agency/`** — Le site de l'agence elle-même.
- **`clients/`** — Un dossier par client, avec sa vitrine HTML, son contexte et ses assets.
- **`_memoire/`** — Mémoire transverse : décisions, leçons, stack technique.
- **`_templates/`** — Modèles réutilisables pour démarrer un nouveau projet rapidement.

## Démarrer un nouveau projet

1. Copier `_templates/NOUVEAU-CLIENT.md` et suivre la checklist.
2. Créer le dossier `clients/0X-nom-client/`.
3. Remplir `CONTEXT.md`, puis bâtir `vitrine.html`.
