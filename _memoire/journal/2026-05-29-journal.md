## Session : Démarrage NEBULA Prospector (agent IA autonome de prospection)

### Contexte
Mongazi veut un agent IA totalement autonome qui :
- Trouve des prospects (PME Afrique de l'Ouest francophone) sur Google Maps + réseaux + annuaires
- Les contacte par email cold (PAS WhatsApp)
- Gère la conversation jusqu'à qualification
- Alerte Mongazi sur **Telegram** uniquement quand le client est prêt à payer
- S'auto-améliore en réécrivant ses propres prompts chaque semaine

### Décisions techniques majeures
- **Stack** : Python 3.11 + Supabase + Claude API + Resend (email) + Telegram Bot. Pas n8n.
- **Sourcing** : Google Maps Places API (officiel, $200/mois gratuit), Jiji, CoinAfrique. Pas de scraping Facebook/Insta (risqué).
- **Outreach** : EMAIL via Resend (pas WhatsApp comme initialement envisagé)
- **Alerte humaine** : Telegram bot (pas WhatsApp, pas email)
- **Volume** : 25 emails/jour max pour préserver la délivrabilité
- **Cibles** : BJ, TG, CI, SN, BF (multi-pays Afrique de l'Ouest francophone)
- **Déploiement** : VPS Hostinger existant (72.61.103.56), systemd + cron

## Session 2 : NOVA + Dashboard temps réel + Analyse de marché

### Demande Mongazi
« Je veux une interface du gens pour voir en temps réel ce que fais mon agent en temps réel, il doit être une véritable entité existant dans le numérique fais carrément tout et analyses le marché »

→ Évolution majeure : transformer le simple agent CLI en **entité numérique avec dashboard live**.

### Décisions
- **Nom de l'entité : NOVA** (choisi par Mongazi parmi NOVA / ZARI / ATLAS / AURA)
- **Stack dashboard** : FastAPI + Jinja2 + Supabase Realtime (pas de framework JS lourd, pas de WebSocket maison)
- **Sécurité** : RLS activée sur `agent_events` + `agent_state` avec policy lecture publique. Le reste des tables reste protégé. La `SUPABASE_ANON_KEY` peut donc être exposée au navigateur sans risque.
- **Design** : thème cosmique (noir profond, nébuleuse violet/cyan/gold), orbe pulsante animée pour matérialiser NOVA
- **Instrumentation** : tous les modules de sourcing (google_maps, jiji, coinafrique) émettent maintenant des events visibles en temps réel sur le dashboard

### Analyse de marché — points clés
- **TAM** des 5 pays cibles (BJ+TG+CI+SN+BF) : ~ 730 000 PME sans site web, ~425 M $/an
- **SAM** : ~110 000 PME avec présence Facebook + email atteignable, ~64 M $/an
- **SOM Y1** réaliste : 100–200 clients, 36–72 M F CFA ARR (~60–120 k $)
- **Blue ocean** : aucun outil global (Apollo, Instantly, Smartlead, lemlist, LGM, Clay) ne couvre ce marché car leur CAC > LTV à 30 000 F CFA/mois. NEBULA peut le servir à coût marginal proche de zéro.
- **Coût opérationnel NOVA** : 100–180 $/mois pour traiter ~2 000 prospects/mois (Anthropic + Google Maps + Hunter + Resend)
- **Secteurs prioritaires 90 jours** : beauté, mode/bijoux, restauration, photo/événementiel, coaching
