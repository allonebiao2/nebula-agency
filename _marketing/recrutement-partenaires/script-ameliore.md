# Script — Vidéo Recrutement Partenaires NEBULA Agency
> Format : TikTok / Facebook Reels | 9:16 | 4K | 60 fps | ~20 secondes  
> Plateforme cible : Bénin · Afrique de l'Ouest francophone  
> Date : 2026-06-22

---

## ANALYSE DE L'ORIGINAL + POINTS D'AMÉLIORATION

**Problèmes du script initial :**
- Trop générique (« étudiant, salarié, entrepreneur » = tout le monde = personne)
- Aucun chiffre concret (taux de commission, taille du marché)
- Pas de dimension cosmique / identité visuelle NEBULA
- CTA flou (« contacte-nous ») sans URL ni action précise
- Durée trop longue pour TikTok (~60 sec lu à voix haute)
- Aucune urgence, aucun FOMO
- Pas de différenciateur vs autres agences

**Choix rédactionnels pour la version améliorée :**
1. Hook ultra-court : question + identité immédiate
2. Chiffre de marché concret pour ancrer l'opportunité
3. Mécanique de commission explicite (35% + réseau N1/N2)
4. Vocabulaire cosmique NEBULA (Météore → Galaxie)
5. URL directe de la plateforme partenaires
6. Durée cible : 18-22 secondes (idéal TikTok/Reels)

## NOTES DE PRODUCTION

### Voix off
- Ton : confiant, direct, pas creux
- Débit : rapide mais posé (public africain urbain, 25-40 ans)
- Langue : français standard + quelques expressions locales possibles

### Musique
- Style : lo-fi trap / afrobeats instrumental minimaliste
- Tempo : ~95-105 BPM pour matcher les scènes
- Volume : -12 dB sous la voix (ne pas couvrir)

### Effets visuels (déjà dans le template HTML)
- Fond cosmos sombre avec nébuleuse violette + cyan
- Particules étoiles animées (260 étoiles twinkle)
- Grille holographique en overlay
- Animations kinétiques mot par mot (translateY + opacity)
- Barre de progression en haut

### Specs techniques
- Résolution : 2160×3840 pixels (4K, 9:16)
- FPS : 60 images/seconde
- Codec : H.264 CRF 18 (qualité quasi-lossless)
- Format export : MP4 vertical

### Pour TikTok
- Durée : 18-22 secondes (algo favorise < 30 sec)
- Sous-titres : OBLIGATOIRES (80% regardent sans son)
- Hashtags suggérés : #Bénin #DigitalAfrique #OpportunitéBénin #NébulAgency #GagnerArgent #Freelance #AfriqueOccidentale

### Pour Facebook
- Même vidéo, couper l'intro si résultats faibles
- Texte du post : reprendre le script comme légende
- Ciblage Meta suggéré :
  - Âge : 18-35 ans
  - Zones : Cotonou, Porto-Novo, Lomé, Abidjan
  - Intérêts : Entrepreneuriat, Digital Marketing, Revenu supplémentaire
  - Comportements : Utilisateurs actifs WhatsApp Business
