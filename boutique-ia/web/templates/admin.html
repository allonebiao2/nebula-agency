Comment-to-DM — quand quelqu'un commente une publication d'une boutique (Page FB / compte IG associé ci-dessus), l'agent lui répond en privé pour démarrer la vente. C'est le seul canal d'acquisition sociale autorisé par Meta : le commentaire vaut consentement. Jamais de message à un inconnu qui n'a rien écrit (cold DM = interdit + bannissement). Une seule réponse privée par commentaire, uniquement sur les Pages associées à une boutique active.

Si un mot-clé est renseigné, l'agent ne répond qu'aux commentaires qui le contiennent (ex : « PRIX », « INFO »). Vide = il répond à tous les commentaires.
