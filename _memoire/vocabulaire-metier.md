## Termes NEBULA

- **Vitrine** — Site one-page HTML autonome livré à un client.
- **SOFIA** — Composant IA livré avec certaines vitrines (à documenter en détail).
- **Couche IA** — Ajout d'automatisation/chatbot par-dessus une vitrine.
- **Brief WhatsApp** — Prise de besoin client via messages WhatsApp.
- **Asset base64** — Image embarquée directement dans le HTML, pas hébergée à l'extérieur.

## Termes techniques

- **HTML autonome** — Un seul fichier `.html` sans dépendance externe.
- **CSS inline** — Styles écrits dans la balise `<style>` du fichier, pas de CSS externe.
- **n8n** — Moteur d'automatisation no-code utilisé pour les workflows IA.
- **Twilio** — Passerelle SMS / WhatsApp.
- **Groq** — Inférence LLM ultra-rapide.
- **Supabase** — Base de données / auth backend.
- **Claude** — LLM Anthropic, raisonnement et génération longue.
- **Gemini** — LLM Google.

## Termes marché

- **WAOF** — West Africa Ouest Francophone (Bénin, Côte d'Ivoire, Sénégal, Togo, Burkina, etc.)
- **Mobile-first africain** — Optimisation pour 3G/4G instable, Android entrée/milieu de gamme.
- **CTA WhatsApp** — Bouton qui ouvre directement une conversation WhatsApp pré-remplie.
