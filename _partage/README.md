# 📂 _partage/ — Dossier d'échange Mongazi ↔ Claude

Dépose ici les captures d'écran, photos, documents, designs ou tout
fichier que tu veux que je voie.

## Comment l'utiliser

### Pour Mongazi
1. Ouvre ce dossier dans l'explorateur Windows :
   `C:\Users\USER\nebula-agency\_partage\`
2. Colle ou glisse-dépose ton image ici
3. Reviens dans Claude Code et dis-moi :
   > *« regarde dans _partage/ »* ou *« ouvre _partage/mon-image.png »*

### Pour Claude
- Je liste le dossier : `Glob _partage/*`
- Je lis l'image avec l'outil Read : `Read _partage/capture.png`

## Bonnes pratiques

- **Nommage clair** : `dashboard-bug-2026-05-30.png` plutôt que `Capture1.png`
- **Pas de secrets** : ne dépose pas ici de captures contenant des mots de
  passe ou clés API (le dossier est suivi par git → tout va sur GitHub)
- **Nettoyage** : tu peux supprimer librement les vieilles captures,
  je les ai déjà analysées

## Astuce capture rapide Windows

`Win + Shift + S` → sélectionne la zone → l'image est dans ton
presse-papiers → ouvre l'explorateur sur `_partage/` → `Ctrl + V` →
elle se sauve automatiquement.
