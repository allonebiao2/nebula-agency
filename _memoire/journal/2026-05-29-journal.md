# Journal — 2026-05-29

## Session : Démarrage NEBULA Prospector (agent IA autonome de prospection)

### Contexte
Mongazi veut un agent IA totalement autonome qui :
- Trouve des prospects (PME Afrique de l'Ouest francophone) sur Google Maps + réseaux + annuaires
- Les contacte par email cold (PAS WhatsApp)
- Gère la conversation jusqu'à qualification
- Alerte Mongazi sur **Telegram** uniquement quand le client est prêt à payer
- S'auto-améliore en réécrivant ses propres prompts chaque semaine

### Décisions techniques majeures
- **Stack** : Python 3.11 + Supabase + Claude API + Resend (email) + Telegram Bot. Pas n8n.
- **Sourcing** : Google Maps Places API (officiel, $200/mois gratuit), Jiji, CoinAfrique. Pas de scraping Facebook/Insta (risqué).
- **Outreach** : EMAIL via Resend (pas WhatsApp comme initialement envisagé)
- **Alerte humaine** : Telegram bot (pas WhatsApp, pas email)
- **Volume** : 25 emails/jour max pour préserver la délivrabilité
- **Cibles** : BJ, TG, CI, SN, BF (multi-pays Afrique de l'Ouest francophone)
- **Déploiement** : VPS Hostinger existant (72.61.103.56), systemd + cron
