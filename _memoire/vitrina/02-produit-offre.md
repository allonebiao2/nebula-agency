# 02 · Produit & Offre

## Ce que Vitrina PROPOSERA (extension d'offre — long terme)
Modules à empiler, chacun monétisable, transformant la vitrine en plateforme :
1. **Catalogue / mini-boutique** (produits + commandes).
2. **Vitrina Social** : posts réseaux générés + planifiés (réutiliser le module Vendora Social).
3. **Kit de marque** : logo + identité générés (déjà amorcé en Business).
4. **Prise de RDV avancée** : agenda, rappels automatiques.
5. **Mini-CRM / fidélité** : fiches clientes, relances, points.
6. **Paiement intégré** : FedaPay quand le volume le justifie.
7. **Mini-analytics** : visites, contacts/RDV générés.
8. **Couche IA** : copie qui VEND + design auto selon le métier (Sonnet texte / Opus créatif).

## Verticales (les autres « portes »)
Beauté (actuel) → mode/couture → resto/maquis → artisans → services. **Même moteur**, mots + photos + thèmes adaptés. On ouvre une porte à la fois, après preuve.
